# KrishiLink
KrishiLink is a modern agriculture marketplace web application where farmers and buyers can connect directly, share crops, agricultural tools, and services.

## Live Site

Server-side-repository:  https://github.com/Muyedrahman/server-krishilink-project

## Features

- **Crop CRUD Operations**: Farmers can add, update, and delete crops.  
- **Interest Tracking**: Buyers can submit their interest in crops, and the status can be updated.  
- **User Profile**: Users can update and retrieve their profile information.  
- **Real-time Database**: All data is stored and accessed in real-time using MongoDB.  
- **RESTful API**: Easy API calls and data exchange through an Express.js backend.
